# Par ou Ímpar 🎲

Jogo de par ou ímpar no terminal feito em Python.

## Como funciona

O jogador escolhe um número de 1 a 10 e aposta se a soma com um número sorteado pelo computador será **par** ou **ímpar**. Se acertar, o jogo continua e a vitória é contabilizada. Se errar, o jogo encerra e exibe o total de vitórias.

## Como executar

Certifique-se de ter o Python instalado, então rode:

```bash
python main.py
```

## Exemplo de partida
Digite um numero de 1 a 10: 7
par ou impar: impar
Soma: 13 → A opção vencedora foi IMPAR
Voce venceu!
Digite um numero de 1 a 10: 3
par ou impar: par
Soma: 9 → A opção vencedora foi IMPAR
Perdeu!
Numero de vitorias: 1
## Tecnologias

- Python 3
- Módulo `random` (nativo)
